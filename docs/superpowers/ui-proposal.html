站点调研总结

🧩

shadcn/ui

⭐ 117K GitHub Stars
完整的设计系统工厂。Button/Input/Card/Dialog/Sidebar/Chart/Form/Toast/Skeleton 全套生产级组件。源码复制到项目，完全控制。自带 Blocks 页面模板和 Charts。

关键发现：不是"组件库"，是"组件工厂"——你拥有源码。支持 GitHub Registry 分发。
基础组件层
✨

React Bits

⭐ 41.8K GitHub Stars
130+ 动画组件。Backgrounds(DotField/Aurora/Beams)、Text Animations(ShinyText/SplitText)、Animations(BlobCursor/MagnetLines/PixelTrail)、Components(Dock)。

关键发现：shadcn registry 兼容 —— 一行命令安装；React 增速最快的动画库。4 种代码口味(JS/CSS, TS/CSS, JS/Tailwind, TS/Tailwind)。
动效增强层
🎪

Uiverse.io

⭐ 7,359 Elements
社区共建 UI 元素库。直接复制 HTML/CSS、Tailwind、React 或 Figma。Buttons/Cards/Loaders/Checkboxes/Toggles/Inputs/Forms/Tooltips/Glassmorphism/Neumorphism。

关键发现：特定元素的宝库——loading spinners、animated toggles、glass cards、创意按钮。
元素灵感源
🎨

Design Prompts

⭐ 30 Design Styles
AI 驱动的设计风格浏览器。30 种风格(Monochrome→Cyberpunk)各有完整预览页+AI Prompt。一键切换 Light/Dark/Sans/Serif/Mono 维度筛选。

关键发现：#05 SaaS Light、#21 Professional Light、#24 Enterprise Light 最匹配客服系统。
风格方向参考
🏆

Awwwards

⭐ 行业设计标准
全球网页设计奖项。Site of the Day (SUI 7.38/10)、Directory/Academy/Market/Collections 生态。

关键发现：提供顶级设计趋势参考和审美基准线，但不直接给组件。Collections 功能可发现同类型产品设计。
设计审美参考
⚡

MotionSites.ai

⭐ Premium Prompts
Premium 动效/动画灵感平台。专注 hero sections 和 motion design 方向。

关键发现：对客服聊天场景直接价值有限，但动画风格参考有用。ReactBits 更实用。
动效审美参考

退换货咨询
退换货流程是什么？
🤖
退换货流程如下：

1. 登录您的账户
2. 进入「我的订单」页面
3. 选择需退换货的订单
4. 点击「申请售后」，填写原因并提交
5. 客服 24 小时内审核

审核通过后寄回商品，仓库签收后 3 个工作日内退款。自签收日起 7 天内可申请无理由退货。
📚 参考：退换货政策.txt · 相似度 92%
