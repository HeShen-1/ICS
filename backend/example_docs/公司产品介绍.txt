公司产品介绍 — 智能客服系统 ICS

一、产品概述
ICS（Intelligent Customer Service）是新一代基于大语言模型的智能客服系统，专为企业提供高效、精准的客户服务解决方案。

二、核心功能
1. 智能问答：基于自建知识库，利用 RAG（检索增强生成）技术，准确回答用户问题。
2. 多渠道接入：支持 Web、微信小程序、企业微信等多种渠道接入。
3. 知识库管理：支持上传 TXT、Markdown、PDF 格式文档，自动解析并向量化入库，支持版本管理和在线更新。
4. 流式对话：采用 SSE（Server-Sent Events）技术，实现打字机效果的实时流式回复。
5. 意图识别：自动识别用户意图，为不同问题类型匹配最优回答策略。
6. 数据分析：实时监控客服数据，包含每日提问量、用户活跃度、知识库使用率等指标。
7. 反馈系统：用户可对回答进行赞/踩评价，帮助持续优化系统表现。

三、技术架构
- 后端：Python FastAPI + SQLAlchemy + MySQL
- 前端：React + TypeScript + Tailwind CSS
- 向量数据库：Milvus Lite（嵌入式模式）
- 大模型：DeepSeek API
- Embedding 模型：BAAI/bge-m3（本地运行）
- 文档解析：LlamaIndex Reader

四、服务版本
- 标准版：支持 50 并发用户，知识库上限 1000 篇文档，适合中小型企业
- 企业版：支持 500+ 并发用户，知识库无上限，支持私有化部署
- 旗舰版：在标准版基础上增加定制化开发、专属运维支持

五、安全保障
- 用户密码使用 bcrypt 哈希存储
- JWT Token 认证，支持 24 小时自动过期
- 所有 API Key 通过环境变量管理
- 每日提问频率限制，防止滥用
- CORS 白名单策略，仅允许授权前端域名
