- 平台功能使用指导（知识库上传、问答配置、会话管理等）

## 3. 常见技术问题

### 3.1 忘记密码怎么办？

如果您忘记了登录密码，可以在登录页面点击"忘记密码"链接，通过注册时绑定的手机号或邮箱接收验证码后重置密码。

具体步骤：
1. 在登录页面点击"忘记密码"
2. 输入注册时使用的手机号或邮箱
3. 点击"发送验证码"，系统将向您的手机或邮箱发送6位数字验证码
4. 输入验证码后，设置新密码（长度不少于6位）
5. 使用新密码登录

### 3.2 文档上传后一直显示"处理中"怎么办？

文档处理时间取决于文档大小和内容复杂度。正常情况下，10MB以内的文档应在30秒内完成处理。

如果超过2分钟仍为"处理中"状态，可能是以下原因：
- 文档格式虽然扩展名正确，但实际内容与扩展名不匹配（如将 .bin 文件改名为 .txt）
- PDF 文件经过加密保护，无法读取内容
- 文档中包含大量图片或特殊字符，解析失败

解决方法：删除该文档后，检查文件内容，重新上传。如仍有问题，请联系技术支持。

### 3.3 AI 回答不准确或不相关？

如果 AI 回答的质量不理想，建议从以下几个方面优化：

1. **丰富知识库内容**：确保知识库文档完整覆盖用户可能问到的各类问题
2. **优化文档质量**：文档内容应结构清晰、表述准确，避免歧义
3. **调整检索参数**：联系技术支持调整相似度阈值（默认0.65）和检索片段数（默认5条）
4. **检查多知识库路由**：确认提问时选择了正确的知识库

### 3.4 如何创建多个知识库？

专业版和企业版用户支持创建多个知识库。操作步骤如下：

1. 进入知识库管理页面
2. 点击"新建知识库"按钮
3. 输入知识库名称和描述（如"产品FAQ"、"售后政策"等）
4. 进入该知识库，上传相关文档
5. 在聊天页面选择对应的知识库，或使用自动路由模式

### 3.5 平台是否支持 API 对接？

企业版用户可使用我们的 RESTful API 将智能问答能力集成到自己的系统中。API 采用 JWT 认证方式，兼容 OpenAI SDK 格式调用。详见开发者文档或联系客户经理获取 API 密钥。
